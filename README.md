symfony-cms-boilerplate
=======================

## Requirements

- PHP 7.0+
- MySQL 5.5+
- Symfony 3.4+
- Bower

## Install
```
$ git clone https://github.com/hiiro/symfony-cms-boilerplate.git
$ cd symfony-cms-boilerplate
$ composer install
$ bin/console d:s:u --force
$ bin/console d:f:l
```
