symfony-cms-boilerplate
=======================

## Requirements

- PHP 7.0+
- MySQL 5.5+
- Symfony 3.4+
- Bower

## Install
```
$ git clone @@@
$ cd symfony-cms-boilerplate
$ composer install
$ bin/console d:s:u --force
$ bin/console d:f:l
```
